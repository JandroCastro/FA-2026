El web component completo

1 · Tres instancias del mismo componente

El mismo <product-card> se usa tres veces con contenido diferente vía slot. El atributo data-stock está observado: cuando cambia, attributeChangedCallback() actualiza el badge y el aspecto visual. Cada card tiene su propio timer interno iniciado en connectedCallback() y parado en disconnectedCallback().
