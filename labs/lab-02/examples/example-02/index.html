De funciones constructoras a clases

1 · La misma entidad, dos sintaxis

Constructor function (ES5) y class (ES6) producen exactamente el mismo objeto. La sintaxis class es sugar syntax: más legible, mismo mecanismo por debajo.

2 · Bajo el capó son lo mismo

Una clase en JavaScript sigue siendo una función. Sus métodos siguen viviendo en el prototipo, no en la instancia.
