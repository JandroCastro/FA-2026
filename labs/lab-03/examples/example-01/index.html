Variables SCSS y design tokens

1 · El compilador borra las variables

Edita el SCSS y pulsa Compilar. Las variables $ desaparecen en el CSS resultante: el compilador sustituye cada variable por su valor antes de que el navegador vea el archivo.

SCSS — tú escribes esto
CSS compilado — el navegador recibe esto
Pulsa "Compilar" para ver el resultado.
Previsualización: compila para ver los estilos aplicados.

2 · CSS custom properties: variables que el navegador sí conoce

Las custom properties (--variable) llegan intactas al navegador y JavaScript puede modificarlas en runtime. Las variables SCSS ya no existen en ese momento: el compilador las sustituyó horas antes. 


          

            

              
Variable SCSS (hardcoded en CSS compilado)

              

              

                background: #6366f1 — fijo, no modificable desde JS
              

            

            

              
CSS custom property (viva en el navegador)

              

              
--demo-primary: #6366f1

            

          

        


        

          
3 · El patrón profesional: los dos sistemas juntos

          

            SCSS variables para valores del build. CSS custom properties para
            valores que el navegador necesita modificar en runtime: temas,
            preferencias del usuario, estados dinámicos.
