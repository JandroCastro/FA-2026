Mobile first y breakpoints

1 · Los estilos base son los de móvil

Arrastra el slider para simular diferentes anchos de viewport. El grid cambia de columnas al cruzar cada breakpoint. Mobile first: los estilos base cubren el móvil y las media queries añaden complejidad progresivamente — nunca la quitan.

340px xs

2 · Mobile first vs Desktop first: mismos breakpoints, dirección opuesta

Ambas estrategias producen el mismo resultado visual. La diferencia está en la dirección de escritura: mobile first añade con min-width; desktop first quita con max-width. El breakpoint activo en el slider de arriba se resalta en cada columna.
