Lab 01 · Example 03

Closures

1 · Estado privado con closure

count vive dentro de createCounter(). No es accesible desde fuera. Solo los métodos devueltos pueden modificarla.

0

counter.value() → 0

Prueba en consola: counter.count → undefined

2 · Dos instancias — dos entornos independientes

Cada llamada a createCounter() crea un closure nuevo. Incrementar uno no afecta al otro.

counterA = createCounter(0)
0
counterB = createCounter(100)
100

counterA y counterB no comparten count. Cada closure tiene el suyo.
